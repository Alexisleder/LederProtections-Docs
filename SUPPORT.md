# Support

Use this file as a quick guide before opening an issue.

---

## Before Reporting a Bug

Please check:

- You are using Minecraft 1.21.x.
- You are using Spigot, Paper, Purpur, or Pufferfish.
- Java 21 is installed.
- WorldEdit is installed.
- WorldGuard is installed.
- Vault is installed if you use economy features.
- PlaceholderAPI is installed only if you use placeholders.

---

## Recommended Debug Command

Run:

```text
/lep info
```

Include the output in your bug report.

---

## What to Include

When reporting a bug, include:

- LederProtections version
- Server software and version
- Minecraft version
- Java version
- WorldGuard version
- WorldEdit version
- Vault version if used
- PlaceholderAPI version if used
- Economy plugin if used
- Output of `/lep info`
- Console errors
- Steps to reproduce the issue

---

## Unsupported Platforms

The following are not supported:

- Folia, for now
- Fabric
- Forge
- NeoForge
- Sponge

Hybrid mod/plugin servers such as Mohist, Magma, or Arclight are not officially supported.
